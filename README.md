# 🌐 AI Website Generator

An AI-powered web application that generates complete websites from simple user descriptions (in English or Hindi).  
Users can preview, edit, and download fully functional websites instantly.

---

## 🚀 Live Demo

👉 Frontend: https://ai-website-generator.vercel.app  
👉 Backend: https://ai-website-generator.onrender.com  

---

## ✨ Features

- 🧠 Generate websites using natural language input
- 🌍 Multilingual support (English + Hindi)
- ⚡ Instant preview with live editing
- 🎯 Pre-built demo templates (Landing, Portfolio, Blog)
- 📦 Download complete website (HTML, CSS, JS)
- 🔄 Dual Mode:
  - Custom Template (Free)
  - OpenAI API (Optional)

---

## ⚠️ Note

AI generation is limited due to API quota.  
You can explore demo outputs using pre-built templates.

---

## 🛠️ Tech Stack

- Frontend: React.js + Tailwind CSS
- Backend: Node.js + Express.js
- AI Integration: OpenAI API (optional)
- Deployment:
  - Frontend → Vercel
  - Backend → Render

## ⚙️ Installation (Local Setup)

### 1. Clone the repository

git clone https://github.com/Suksham-123/ai-website-generator.git  
cd ai-website-generator  

---

### 2. Backend Setup

cd backend  
npm install  
node index.js  

---

### 3. Frontend Setup

cd frontend  
npm install  
npm start  

---

## 🔑 Environment Variables (Optional)

Create a `.env` file in backend:

OPENAI_API_KEY=your_api_key_here
